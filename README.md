# Data-Science_Customer-Segmentation-Using-K-Means-Clustering
Proyek Data Science yang menerapkan metode K-Means Clustering untuk melakukan segmentasi pelanggan berdasarkan karakteristik dan perilaku belanja. Segmentasi ini digunakan sebagai dasar penyusunan strategi pemasaran yang lebih tepat sasaran.

Objective: Mengidentifikasi kelompok pelanggan berdasarkan pola pendapatan dan perilaku belanja.
Process: Melakukan analisis dan segmentasi menggunakan metode Unsupervised Learning (K-Means Clustering) dengan Elbow Method untuk menentukan jumlah cluster optimal.
Result: Dihasilkan empat segmen pelanggan dengan karakteristik berbeda yang dapat digunakan untuk strategi pemasaran dan pengambilan keputusan bisnis.
